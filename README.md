# Adventure3d

This is a work-in-progress 3d game.

![Preview](/screenshots/preview.png)

## Build and Run

```bash

    # Make sure you have GLFW installed (X11)
    sudo pacman -S glfw

    ./build.sh
    ./adventure

```

## Controls

```

    w = Move Forward
    a = Strafe Left
    s = Move Backwards
    d = Strafe Right

    space = Jump

    tab = toggle wireframe
    m = Toggle camera mode

```
